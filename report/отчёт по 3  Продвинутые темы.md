**Задание 5:** Скрипт для нахождения НОД.

![3.4.5](https://github.com/slavagoltzoni-boop/linux-course-report/blob/main/screenshots/3.%20Продвинутые%20темы/3.4%20Скрипты%20на%20bash%20разное/5%20задание/3.4.5.png)

**Ответ:**
```bash
#!/bin/bash
gcd() {
    local m=$1
    local n=$2
    if [[ $m -eq $n ]]; then
        echo "GCD is $m"
    elif [[ $m -gt $n ]]; then
        gcd $((m - n)) $n
    else
        gcd $m $((n - m))
    fi
}
while true; do
    read m n
    if [[ -z $m ]]; then
        echo "bye"
        exit 0
    fi
    gcd $m $n
done
```
**Задание 6:** Калькулятор на bash.

![3.4.6](https://github.com/slavagoltzoni-boop/linux-course-report/blob/main/screenshots/3.%20Продвинутые%20темы/3.4%20Скрипты%20на%20bash%20разное/6%20задание/3.4.6.png)

**Ответ:**
```bash
#!/bin/bash
while [[ True ]]; do
  read birinchi amal ikkinchi
  if [[ $birinchi == "exit" ]]; then
    echo "bye"
    break
  elif [[ "$birinchi" =~ "^[0-9]+$" && "$ikkinchi" =~ "^[0-9]+$" ]]; then
    echo "error"
    break
  else
    case $amal in
      "+") let "result = birinchi + ikkinchi";;
      "-") let "result = birinchi - ikkinchi";;
      "/") let "result = birinchi / ikkinchi";;
      "*") let "result = birinchi * ikkinchi";;
      "%") let "result = birinchi % ikkinchi";;
      "**") let "result = birinchi ** ikkinchi";;
      *) echo "error" ; break ;;
    esac
    echo "$result"
  fi
done
```
Пояснение: Калькулятор с поддержкой операций +, -, *, /, %, ** и командой exit. При некорректной операции выводит "error" и завершает работу.

- `chmod a+wx file.txt; chmod o-wx file.txt; chmod g-x file.txt`

**Пояснение:** `764` = rwx(7) rw(6) r(4). Остальные последовательности дают тот же результат.

**Вопрос 3:** Что можно посчитать командой `wc`?

![3.7.3](https://github.com/slavagoltzoni-boop/linux-course-report/blob/main/screenshots/3.%20Продвинутые%20темы/3.7%20Разное/3%20задание/3.7.3.png)

**Ответ:**
- Количество символов
- Количество слов
- Длину самой длинной строки

**Пояснение:** `wc` считает строки, слова и символы. С опцией `-L` — длину самой длинной строки. Количество предложений и конкретных букв `wc` не считает.

**Вопрос 4:** Команда для вывода размера текущей директории.

![3.7.4](https://github.com/slavagoltzoni-boop/linux-course-report/blob/main/screenshots/3.%20Продвинутые%20темы/3.7%20Разное/4%20задание/3.7.4.png)

**Ответ:** `du -sh .`

**Пояснение:** `-s` — общий размер, `-h` — удобный для чтения формат, `.` — текущая директория.

**Вопрос 5:** Самая короткая команда для создания трёх директорий.
